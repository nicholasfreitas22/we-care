# 🐶 We Care About Your Pet

Projeto de landing page desenvolvido para praticar HTML e CSS durante meus estudos no DevClub.

A página apresenta um layout moderno para um serviço de cuidados com pets, com foco em responsividade e design limpo.

<img src="https://github.com/nicholasfreitas22/we-care/blob/main/eeeee.png?raw=true">

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

✔️ Layout moderno  
✔️ Menu de navegação  
✔️ Botão de login  
✔️ Design responsivo  
✔️ Seção de apresentação do serviço

## 📚 Aprendizados

Neste projeto eu pratiquei:

- Estruturação semântica com HTML
- Estilização com CSS
- Organização de layout de landing page
